À exécuter une seule fois
Les nouvelles fonctions (sous-tâches IA, estimation d'effort, journal Stand-up) ont besoin de colonnes/tables supplémentaires. Ouvrez Supabase → SQL Editor, collez ce script et exécutez-le. Aucune donnée existante n'est touchée. Tant que ce n'est pas fait, l'app reste 100% fonctionnelle ; ces fonctions s'activeront ensuite automatiquement.
